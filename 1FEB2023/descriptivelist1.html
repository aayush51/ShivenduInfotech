Descriptive List

HTML Tags With Their Descriptions

Html tag-
It is the root element of an HTML document.

Head tag-
It is used to contain all the head elements.

Body tag-
It is used to contain all the visible page content.

Title tag-
It is used to specify a title for the document.

H1 tag-
It is used to define a large heading.

H2 tag-
It is used to define a medium heading.

H3 tag-
It is used to define a small heading.

H4 tag-
It is used to define a smaller heading.

H5 tag-
It is used to define a smaller heading.

H6 tag-
It is used to define a smaller heading.

P tag-
It is used to define a paragraph.

Br tag-
It is used to insert a single line break.

Hr tag-
It is used to insert a horizontal rule.

Img tag-
It is used to insert an image.

Table tag-
It is used to define a table.

Tr tag-
It is used to define a table row.

Td tag-
It is used to define a table data cell.

Th tag-
It is used to define a table header cell.

Ul tag-
It is used to define an unordered list.

Ol tag-
It is used to define an ordered list.

Li tag-
It is used to define a list item.

Dl tag-
It is used to define a descriptive list.

Dt tag-
It is used to define a term/name in a description list.

Dd tag-
It is used to define a description of a term/name in a description list.

Form tag-
It is used to define an HTML form for user input.

Input tag-
It is used to define an input control.

Textarea tag-
It is used to define a multiline input control (text area).

Select tag-
It is used to define a drop-down list.

Option tag-
It is used to define a selectable item in a drop-down list.

Button tag-
It is used to define a clickable button.

Label tag-
It is used to define a label for an input control.

Fieldset tag-
It is used to group related elements in a form.

Legend tag-
It is used to define a caption for a fieldset element.

Link tag-
It is used to define the relationship between a document and an external resource (most used to link to style sheets).

Style tag-
It is used to define style information for a document.

Script tag-
It is used to define a client-side script (JavaScript).

Meta tag-
It is used to provide metadata about an HTML document.

Div tag-
It is used to define a section in a document.

Span tag-
It is used to group inline-elements in a document.
